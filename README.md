# `ytdl`

## Requirements

* `redis`
* `ruby`
* `bundler`

## Development

## Installing gems

```console
bundle install
```

## Installing packages

### FreeBSD

```console
pkg install redis
```

## Stating the application

```console
bundle exec foreman start
```

## Running tests

```console
bundle exec rake test
```

## Writing tests

See:

* [rspec-html-matchers](https://github.com/kucaahbe/rspec-html-matchers)
* [rack-test](https://github.com/rack/rack-test)
